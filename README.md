# datadog-cloudformation-macro

This repository contains Datadog CloudFormation macros to use with raw CloudFormation templates or SAM/CDK deployments.

* [Datadog Serverless Macro](https://github.com/DataDog/datadog-cloudformation-macro/tree/master/serverless): installs Datadog Lambda Library to Python, Node.js, .NET, Ruby, and Java Lambda functions to collect custom metrics and traces
